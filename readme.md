## 有关bonus
### bonus5.1 图片裁剪
在这次的pj中，我实现图片裁剪的方法是将图片放置在一个容器中，设置容器的大小为页面所需要的大小，在将其overflow设置为hidden；即可实现对图片视觉上的切割

### bonus5.2 响应式布局
在这次pj中，我设计网页遵循的布局是流式布局；即在宽度上以百分比进行布局划分；在收缩的时候，其页面的基本布局不会发生明显变化，只有少数换行的情况
