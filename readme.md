## 有关bonus
### bonus5.1 图片裁剪
因为在这次pj中，宽度我是使用百分比进行设置，所以在裁剪上我是利用jQuery设置宽高比来实现

### bonus5.2 响应式布局
在这次pj中，我设计网页遵循的布局是流式布局；即在宽度上以百分比进行布局划分；在收缩的时候，其页面的基本布局不会发生明显变化，只有少数换行的情况
